# Document-Scanner-using-OpenCV
This project was developed as part of Minor Project II during the 7th semester at the University.

Installation & Usage
1. Download or clone this repository.
2. Open the project in Visual Studio Community.
3. Download & install: https://github.com/opencv/opencv/releases/download/4.12.0/opencv-4.12.0-windows.exe
4. Copy "opencv" folder from the installation to the "Liabraries" directory in project.
5. Build the solution and run the application.
